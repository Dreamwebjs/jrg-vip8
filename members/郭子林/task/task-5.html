样式有几种引入方式?

  1. 行内式：在标签的style属性内设置css样式。
  2. 嵌入式：写在页面的标签内，单个页面还行，页面多时，不易维护。
  3. 连接式<link rel="stylesheet" type="text/css" href="mystyle.css">推荐使用。
  4. 导入式@important “mystyle.css” 。

link 和 @import有什么区别

  1. link引用CSS时，在页面载入时同时加载；@import需要页面网页完全载入以后加载。
  2. link是XHTML标签，除了加载CSS外，还可以定义RSS等其他事务；@import属于CSS范畴，只能加载CSS。
  3. link是XHTML标签，无兼容问题；@import是在CSS2.1提出的，低版本的浏览器不支持。
  4. link支持使用Javascript控制DOM去改变样式；而@import不支持。

文件路径../main.css 、./main.css、main.css有什么区别

  1. ../main.css指的是上一级文件夹下的main.css。
  2. ./main.css当前文件夹下的main.css，效果与main.css一样。
  3. main.css当前文件夹下的main.css，效果与./main.css一样。

px、em、rem分别是什么？有什么区别？如何使用

  1. px:像素，用PX设置字体大小时，比较稳定和精确将，可以获得失真最小的一致性。然而网站访客在缩放浏览器时原有的网页布局有可能会被打破。
  2. em:根据基准来缩放字体的大小。EM实质是一个相对值，而非具体的数值。em的问题是字体大小会随着层级叠加。例如li的字体大小是15px，但li里的li则会因为1.25*15px而叠加到18.75px，若再有一层li，则会进一步1.25*18.75叠加上去……。
  3. rem:rem的字义是「root em」，简单地说，就是先宣告一个所有网页元件都共用的em单位：html { font-size:75% } /* 16px*75%=12px */接着，所有字体大小都依此标准来设计制定：body { font-size:1rem } /* 1*12px=12px */h1 {font-size:2rem} /* 2*12px=24px */等等。
