New Semantic/Structural Elements(新的语义/结构元素)

HTML5 offers new elements for better document structure:(HTML5提供了更好的文档结构的新元素)

Tag(标签) Description（描述）
< article > Defines an article in the document（在文档中定义一个文章）
< aside > Defines content aside from the page content（从页面内容定义内容）
< bdi > Defines a part of text that might be formatted in a different direction from other text（ 定义文字的一部分可能在从其他文本不同的方向进行格式化）
< details > Defines additional details that the user can view or hide（定义其他详细信息，用户可以查看或隐藏）
< dialog > Defines a dialog box or window（定义对话框或窗口）
< figcaption > Defines a caption for a < figure > element（定义一个标题为< figure >元素）
< figure > Defines self-contained content, like illustrations, diagrams, photos, code listings, etc.( 定义自包含的内容，如插图，图表，照片，代码清单等。)
< footer > Defines a footer for the document or a section(定义页脚的文档或节)
< header > Defines a header for the document or a section(定义标头的文件或部分)
< main > Defines the main content of a document(定义文档的主要内容)
< mark > Defines marked or highlighted text(定义标注或突出显示的文本)
< menuitem > Defines a command/menu item that the user can invoke from a popup menu(定义了用户可以从一个弹出式菜单调用命令/菜单项)
< meter > Defines a scalar measurement within a known range (a gauge)[定义了一个已知的范围内的标量测定（量规）]
< nav > Defines navigation links in the document(定义在文件中导航链接)
< progress > Defines the progress of a task(定义了一个任务的进度)
< rp > Defines what to show in browsers that do not support ruby annotations(定义在不支持红宝石注释的浏览器中显示的)

< ruby > Defines a ruby annotation (for East Asian typography)[定义红宝石注释（东亚版式）]
< section > Defines a section in the document(定义文档中的一个部分)
< summary > Defines a visible heading for a < details > element(定义可见标题为< details >元素)
< time > Defines a date/time(定义日期/时间)
< wbr > Defines a possible line-break(定义可能换行)

HTML5 - New Attribute Syntax(HTML5-新的属性语法)

HTML5 allows four different syntaxes for attributes.(HTML5允许四种不同的语法属性:)

This example demonstrates the different syntaxes used in an < input > tag:(这个例子演示了在< 输入 >标签的不同语法形式：)

Tag Example
Empty（空） < input type="text" value="John" disabled >（disabled 禁用）
Unquoted（未加引号） < input type="text" value = John>
Double-quoted（双引号） < input type="text" value="John Doe" >
Single-quoted（单引号） < input type="text" value='John Doe' >

In HTML5, all four syntaxes may be used, depending on what is needed for the attribute.(在HTML5，所有四个语法可以使用，这取决于什么是属性的需要。)
