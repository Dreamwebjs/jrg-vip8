.clearfix原理

前端

首先要明白：
1. 一个 div 的高度，是由其内部的文档流的高度决定的
2. 浮动元素是脱离标准文档流的，所以会致使父元素的高度坍塌
3. 这个时候需要使用清除浮动

清楚浮动的方法：

方法一：
在父元素的内容的末尾，添加一个空的 div，使用 clear:both 致使两旁不能有浮动，从而这个 div 将会另起一行，又因为这个最后的 div 为空，所以没有高度，所以可以使父元素包含前面的浮动元素，达到清楚浮动的效果

  1. <!DOCTYPE html>
  2. <html lang="en">
  3. <head>
  4.     <meta charset="UTF-8">
  5.     <title>Document</title>
  6. </head>
  7. <body>
  8.     <ul>
  9.         <li>1</li>
  10.         <li>2</li>
  11.         <li>3</li>
  12.         <div style="clear: both;"></div>
  13.     </ul>
  14. </body>
  15. </html>

方法二：
在第一种的方法上，可以使用 after 伪类进行改进。使用after伪类和内容声明在现在内容的末尾添加新的内容。

  1. div:after{
  2.     content:"";
  3.     display:block;
  4.     clear:both;
  5. }

方法三：
因为方法二在同时清除多个浮动的时候不便捷，所以还可以改进。使用.clearfix方法，通过类可以叠加，为父元素添加.cleafix的类名

  1. .cleafix:after{
  2.     content:"";
  3.     display:block;
  4.     clear:both;
  5. }
